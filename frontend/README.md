# Luxe E-Commerce Frontend

This is a modern, responsive, glassmorphism-themed React application built with Vite to seamlessly integrate with the Django backend.

## 🚀 Beautiful Features

- **Stunning UI**: Premium glassmorphism design with vibrant gradients, sleek micro-animations, and modern typography (Inter/Outfit).
- **Authentication**: JWT-based login and registration flow integrated with Django securely.
- **Product Discovery**: Browse beautiful product listing grids with image hover effects.
- **AI Recommendations**: View personalized "You Might Also Like" AI suggestions integrated into product details.
- **Shopping Cart**: Real-time cart updates with dynamic total calculations.
- **Secure Checkout**: Streamlined checkout experience.
- **Order Tracking**: Beautiful dashboard to track past and current orders.

## 🛠️ Requirements
You **MUST** have Node.js installed on your machine to run this frontend.
Download the LTS version here: https://nodejs.org/

## 🏃‍♂️ How to Run

1. Open a terminal and navigate to this folder:
   ```bash
   cd frontend
   ```
2. Install all dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and go to the link shown in the terminal (usually `http://localhost:5173/`).

*Make sure your Django Backend (`e_commerce`) and Docker containers are running first so that the APIs connect properly!*
